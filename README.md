# 🌍 Gram Suvidha - Rural Information Portal
**Empowering Villages through Offline-First Digital Access**

## 🚀 The Problem
Rural communities often face "Information Poverty." Even with smartphones, farmers and students struggle with:
1. **Slow Internet:** Traditional websites fail to load on 2G/3G networks.
2. **Scams:** Fake links for government schemes often mislead users.
3. **Complexity:** Most portals are too cluttered for non-technical users.

## 🛠️ The Solution
Gram Suvidha is a **Progressive Web App (PWA)** designed to be a one-stop shop for village needs. It provides direct, verified links to Agriculture, Education, and Healthcare services.

### Key Features:
- **Offline Mode:** Works without internet once loaded (using Service Workers).
- **Direct Support:** One-tap WhatsApp/Call integration for community help.
- **Icon-Driven UI:** Simplified navigation for all literacy levels.
- **Mobile Optimized:** Built with Bootstrap for a seamless mobile experience.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **App Logic:** JavaScript (PWA Service Workers)
- **Deployment:** GitHub Pages
- **Configuration:** Manifest.json for Android/iOS installation

## 📈 Social Impact
This project aims to bridge the digital divide by providing verified information to the last mile. It ensures that a farmer in the field has the same access to Mandi rates as a trader in the city.
