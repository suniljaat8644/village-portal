# Gram Suvidha Portal 🚜🎓🏥
### "Bridging the Digital Divide, One Village at a Time."

**Gram Suvidha** is a Progressive Web App (PWA) designed to bring essential government services, agricultural data, and educational information to rural communities in India. Built with a focus on accessibility and low-connectivity environments.

---

## 🚀 Key Features

* 🚫 **Offline Access (PWA):** Works without active internet, caching critical pages for offline use.
* 🎤 **Voice Search (A11y):** Multilingual (Hindi/English) voice search for users who find typing difficult.
* 🔔 **Live Updates:** Integrated with Firebase Cloud Messaging for real-time scheme notifications.
* 🌾 **Agriculture Hub:** Live Mandi rates simulation and localized weather advice.
* 🏥 **Healthcare Locator:** One-touch Emergency (108) call and GPS-enabled "Hospitals Near Me" locator.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (ES6+)
* **Backend/PWA:** Service Workers, Cache API, Web Speech API
* **Cloud Services:** Firebase Cloud Messaging (Notifications), GitHub Pages (Hosting)
